Tasks

  • Exercise 1

    This is a webpage that displays information and images about two different artists, Vincent van Gogh and Caravaggio, along with their famous artworks. The webpage uses HTML and CSS to create a split layout with two sections, each containing an image, a title, and a description of the artist and their art style.

  • Exercise 2

    The exercise was targeted towards the understanding of JavaScript. It creates a webpage with multiple sections and a div, each containing paragraphs and lists. The purpose of the exercise being able to apply styling changes to specific elements within the HTML.

  • Exercise 3

    This website serves as a practice exercise, expanding knowledge of HTML, CSS, and JavaScript. It features interactive elements like a button to make text disappear, options to change font styles, a slider to adjust box width, and a dropdown menu to display different images.
